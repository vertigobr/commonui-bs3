Descrição

O Sistema Urubu foi lançado em abril de 2014 e atualmente (Julho de 2017) temos mais de 21 mil usuários em todo o Brasil. Qualquer pessoa pode fazer parte da nossa rede de conservação da biodiversidade. Basta fazer o download do aplicativo (Urubu Mobile), se cadastrar e toda vez que encontrar um animal atropelado tirar uma foto com o próprio aplicativo. Essa foto será georeferenciada e saberemos quem foi a pessoa que nos enviou.

O Sistema já reuniu mais de 20 mil registros de atropelamentos e está atualizando sua base com dados oriundos de outros segmentos da sociedade, tais como os dados de licenciamento ambiental.

SERVIÇOS

Urubu Mobile

É a principal forma de colaborar com o Sistema Urubu. É um aplicativo gratuito que permite qualquer pessoa registrar informações de atropelamento de fauna silvestre. Se você quer colaborar, baixe o Urubu Mobile (disponível para Android e iOS), se cadastre e faça parte da nossa rede.

Urubu Web

O Urubu Web é o cérebro do Sistema Urubu. Aqui os analistas do CBEE recebem e gerenciam todas as informações. Cada usuário tem um nível de acesso diferente. Se você enviou fotos para o Sistema Urubu e deseja saber se elas foram recebidas e validadas, acesse o Urubu Web com seu login e senha.

Urubu Map

O Urubu Map permite a visualização de todos os dados públicos do Sistema Urubu no mapa do Brasil. É possível verificar a distribuição dos dados, gerar mapas com outras informações ecológicas, realizar análises simples e visualizar informações de cada foto enviada para o Sistema.

Urubu Info

No Urubu Info é possível visualizar tabelas com os dados públicos do Sistema Urubu. Elas são atualizadas diariamente e apresentam o número de atropelamento registrados por classe, estado, unidades de conservação, entre outros.
