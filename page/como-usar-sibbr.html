Como usar o SiBBr

Entenda aqui como você pode usar o SiBBr e como você pode participar e contribuir para o aumento do conhecimento sobre a biodiversidade brasileira.

Dados e informações são fornecidos por um conjunto de organizações, sites, indivíduos, redes de pesquisa, comunitários, departamentos governamentais e sistemas de informação que compartilham seus dados através do SiBBr contribuindo para o conhecimento global sobre a biodiversidade e, consequentemente, para as soluções que promoverão sua conservação e uso sustentável. Como fazer análises espaciais usando os dados do SiBBr

O portal espacial pode acessado pelo menu Busque e analise > portal espacial (http://sibbr-portal-espacial.rnp.br/)

Legenda

1 Adicionando dados
1.1 Espécies

Para adicionar uma espécie, vá até o menu Adiciona > espécies e selecione a espécie desejada, nesse exemplo foi adicionada a camada de ocorrências para a classe insecta.

Legenda

Legenda

1.2 ?
1.3 Áreas

Para selecionar áreas , como por exemplo Unidades de conservação, vá até o menu Adiciona > área .

Legenda

1.4 Camadas

Para adicionar uma camada, vá até o menu Adiciona > camada. Em uma lista pré definida, selecione a camada que se deseja colocar no mapa.

1.5 Busca por atributos

Para filtrar por atributos, vá até o menu Adiciona > facet.
Com essa funcionalidade é possível selecionar um atributo único de uma espécie em relação a todos os registro de espécies do banco de dados.Assim deve-se escolher a área que se deseja aplicar a seleção e em seguida selecionar o atributo (termos do darwin core) de interesse , por exemplo família, espécie, gênero etc.
