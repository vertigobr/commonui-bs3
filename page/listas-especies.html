Listas de espécies são inventários de espécies ou outros taxa estabelecidas com base em alguns critérios, principalmente taxonômicos ou associados a uma localidade. Entre vários tipos, destacamos listas de espécies ameaçadas no mundo (red lists), no Brasil ou por estado, lista de espécies exóticas e/ou invasoras, endêmicas e sensíveis de uma determinada região, bem como listas de observação e avistamento de espécies em unidades de conservação, parques, etc. As listas são bons indicadores do estado de conservação permitindo conhecer a biodiversidade local por grupos taxonômicos.

Confira o passo a passo para publicação de uma lista de espécie e entre em contato com o SiBBr no e-mail captacao@sibbr.gov.br em qualquer momento.

  1. Cadastre sua instituição

    Para iniciar o processo de publicação é necessário cadastrar a instituição associada ao conjunto de dados como instituição publicadora do SiBBr. Verifique se sua instituição encontra-se na
    Lista de instituições parceiras do SiBBr. Caso contrário, preencha o formulário de cadastro.

  2. Padronize seus dados ao padrão Darwin Core

    Para fins de publicação e compartilhamento de dados, estes precisam estar padronizados em Darwin Core (DwC). O Padrão Darwin Core é um padrão internacional para dados sobre biodiversidade que inclui um glossário de termos ou títulos das colunas de uma planilha permitindo que seus dados sejam compreendidos e utilizados por qualquer pessoa em qualquer lugar.

    Para mais informações sobre o padrão Darwin Core clique na página oficial e assista o vídeo Estruturação e padronização de dados em Darwin Core para publicação no SiBBr. Utilize a planilha em Excel (.xls) ou a planilha em LibreOffice (.ods) do SiBBr com o cabeçalho em Darwin Core para estruturar seus dados.

  3. Melhore a qualidade dos dados

    Verifique e melhore a qualidade dos dados mediante ferramentas online do SiBBr e de outras plataformas internacionais. Identifique possíveis erros de digitação e de formato bem como corrija falhas na taxonomia e incoherencias geográficas do seu conjunto de dados.

    Ferramentas de qualidade de dados

    • HarpiaTax - Dicionário Taxonômico
    • RivulusValida - Validador de nomes científicos
    • TungaGeo - Conversor de coordenadas
    • Sandbox
    • Validador do Darwin Core Arquive (DwC-A) - GBIF.

    Para mais informações sobre qualidade de dados em biodiversidade acesse

    • Introdução à qualidade de dados de biodiversidade
    • Princípios de qualidade de dados

  4. Entre em contato com o SiBBr

    Envie um e-mail para captacao@sibbr.gov.br com algumas informações sobre seu conjunto de dados.

    • Nome completo
    • Instituição ou rede de pesquisa associada
    • Breve descrição do conjunto de dados (título e resumo)
    • Especificações técnicas sobre o conjunto de dados (software de gerenciamento de dados e formato da planilha)

  5. Suporte

    A equipe de captação de dados do SiBBr será responsável pelo suporte durante todo o processo de publicação. Revisaremos o conjunto de dados identificando inconsistências e sugerindo mudanças para melhorar a completude e qualidade dos dados. Esta é a etapa mais importante, que inclui desde a criação do conjunto de dados na ferramenta de publicação, a carga dos e a posterior visualização dos dados no portal.

  6. Selecione a licença e complete os metadados

    Entraremos em contato via e-mail com nome de usuário e senha de acesso à ferramenta de publicação. Nos metadados deves preencher todos as informações solicitadas de forma clara e clara. Lembre-se de colocar os contatos do recurso, os métodos de amostragem e outras informações relevantes. Selecione uma das três licenças de uso dos dados do SiBBr.

  7. Registre no GBIF (opcional)

    Uma vez publicado o conjunto de dados no SiBBr, o publicador escolhe se deseja registrar o conjunto de dados também na Plataforma Global de Informação sobre Biodiversidade (GBIF, www.gbif.org).
