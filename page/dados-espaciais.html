Dados espaciais

Conteúdo da página:

  • Introdução
  • Cadastre sua instituição
  • Entre em contato com o SiBBr
  • Suporte e publicação

Dados espaciais também chamados de dados geográficos ou geoespaciais são todos os dados que podem ser associados a algum local. Estes dados descrevem (representam) objetos, fatos e fenômenos da superfície terrestre associados à sua localização sobre a Terra, em um certo instante ou período de tempo.

Veja os dados espaciais e camadas ambientais disponibilizados no Portal Espacial do SiBBr onde encontrará todo tipo de shapes, camadas e outras informações ambientais onde, além de visualizar os registros de ocorrência das espécies, bases hidrográficas, rodovias, uso do solo entre outras, é possível realizar diversas análises como:

  • Calcular área de distribuição de uma espécie
  • Elaboração de gráficos de dispersão de uma variável
  • Comparação de áreas
  • Gerar relatório oara uma área selecionada
  • Gerar tabelas com base em camadas selecionadas

Para ter acesso aos metadados dos dados disponibilizados no Portal Espacial do ALA, acesse o Catálogo de metadados do SiBBr Infraestrutura de Dados Espaciais do SiBBr (IDE)

Confira o passo a passo para publicação de dados espaciais e entre em contato com o SiBBr no e-mail captacao@sibbr.gov.br em qualquer momento.

  1. Cadastre sua instituição

    Para iniciar o processo de publicação é necessário cadastrar a instituição associada ao conjunto de dados como instituição publicadora do SiBBr. Verifique se sua instituição encontra-se na
    Lista de instituições parceiras do SiBBr. Caso contrário, preencha o formulário de cadastro.

  2. Entre em contato com o SiBBr

    Envie um e-mail para captacao@sibbr.gov.br com algumas informações sobre seu conjunto de dados.

    • Nome completo
    • Instituição ou rede de pesquisa associada
    • Breve descrição dos dados
    • Especificações técnicas o conjunto de dados (formato)

  3. Suporte e publicação

    A equipe de captação de dados do SiBBr será responsável pelo suporte durante todo o processo de publicação. Revisaremos o conjunto de dados identificando inconsistências e sugerindo mudanças para melhorar a completude e qualidade dos dados. Esta é a etapa mais importante, que inclui desde a criação do conjunto de dados na ferramenta de publicação, a carga dos e a posterior visualização dos dados no portal espacial.
