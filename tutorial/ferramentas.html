Relatório de área

Uma vez com uma área definida seja desenhando o polígono, seja importando um shapefile de polígono , é possível pedir um relatório para essa área.

O relatório pode ser interativo (neste tipo o foco é mostrar as ocorrências de espécies e grupos na área ) ou detalhado eu pode ser baixado no formato PDF ,onde é levantado além das informações de espécies , as demais bases de dados (disponíveis pelo menu Adicionar bases de dados) que estão contidas na área definida.

Selecionando a opção Relatório de área – interativo , a tela seguinte pede que o usuário selecione a área que desenhou ou a área importada ou ainda pode pedir o relatório para o Brasil. Clique em próximo para obter o relatório:

A tela seguinte mostra o relatório para a área , como valor da área , número de espécies, número de espécies endêmicas entre outros.

Calcular área de ocupação (AOO) e extensão de ocorrência (EOO)

Com base em uma solicitação de espécies pelo menu Adicionar > Espécies é possível obter as áreas de ocupação (AOO) de e extensão de ocorrência (EOO) para uma espécie. A metodologia utilizada para calculo no ALA é descrita no link ALA .

Por exemplo, fazendo uma busca por lobo – guará , o portal apresentará as ocorrências no portal espacial:

Para obter a áreas de ocupação (AOO) de e extensão de ocorrência (EOO) para essa espécie , basta ir no menu Ferramentas > Calcular área de ocupação (AOO) e extensão de ocorrência (EOO):

Na tela seguinte , o usuário escolhe alguns parâmetros:

  1. Selecione a área de interesse: Por padrão a opção de área atual (Current extent ) fica marcada;
  2. Selecione a espécie : Para esse tutorial foi utilizada a espécie lobo – guará.
  3. AOO grid resolution : Tamanho do grid a se usado para a área de ocupação. O padrão é 2 Km.
  4. O alfa hull a ser usado para determinar a área para inclusão ao calcular a EOO.

Uma vez preenchido os campos clique em próximo para ver o resultado:

Os resultados são:

  • O número de registros usados para os cálculos;
  • Os taxóns utilizados;
  • Área de ocupação (AOO: m²);
  • Extensão da ocorrência (EOO: m²);
  • Nova camada pontual da ocorrência de táxons (shapefile);
  • Nova camada de área: AOO (shapefile);
  • Nova camada de área: EOO (mínimo polígono convexo - shapefile);
  • Nova camada de área: alpha hull (shapefile)
